# Proyecto 1 (Full) TranxMex - Análisis Operativo

## Resumen del proyecto

TranxMex es un proyecto de análisis de datos en Python enfocado en limpiar, organizar y convertir transacciones brutas en información útil para la toma de decisiones. El objetivo principal es entender mejor el comportamiento operativo y financiero del negocio a partir de una base de datos desordenada.

## Qué problema resuelve

El proyecto parte de la idea de que los datos por sí solos no explican el negocio si no se depuran e interpretan correctamente. Por eso, se trabajó primero en la calidad de la información y después en el análisis operativo para identificar dónde se concentra la actividad, qué empresas o categorías tienen más peso y qué diferencias aparecen por país o canal.

## Metodología

El flujo de trabajo sigue una secuencia clara: limpieza de datos, validación de estructura, tratamiento de nulos y duplicados, normalización de moneda y revisión de columnas clave como fecha, importe, estado y categoría. Después de esa base confiable, se realiza un análisis exploratorio para detectar patrones, métricas relevantes e insights de negocio.

## Resultados

El resultado final incluye un dataset limpio, reportes de análisis, visualizaciones y un dashboard ejecutivo. El proyecto destaca especialmente el comportamiento de los estados de transacción, la evolución mensual, la distribución por empresa, canal de pago, categoría y país.

## Valor del proyecto

TranxMex transforma datos transaccionales en una visión más clara del estado operativo del negocio. Esto permite detectar fricciones, entender el flujo de capital y apoyar decisiones con evidencia en lugar de suposiciones.

# Proyecto 2 (Full) TranxMex - Optimización Financiera

## Resumen del proyecto

TranxMex es un proyecto de análisis y optimización financiera desarrollado con datos transaccionales, dashboards y herramientas de visualización como Excel y Power BI. Su objetivo es convertir información dispersa en una narrativa clara para apoyar decisiones ejecutivas.

## Qué problema resuelve

El proyecto parte de una situación de iliquidez evidente en la empresa y busca entender qué ocurre, dónde ocurre, con quién ocurre y por qué ocurre. El análisis se centra especialmente en transacciones con estados como pendiente, cancelado y negado, porque reflejan el estado financiero-operativo de TranxMex.

## Metodología

Primero se modelaron y organizaron los datos para construir una base útil de análisis. Luego se definieron KPIs, se diseñaron dashboards en Excel y Power BI, y se preparó una presentación en PowerPoint para comunicar hallazgos, soluciones y acciones concretas al nivel ejecutivo.

## Entregables

El proyecto incluye un dataset limpio, un dashboard en Excel, un dashboard en Power BI, una presentación ejecutiva y documentos con gráficas, métricas DAX e insights accionables. Todo está orientado a mostrar el estado de la empresa de forma visual, resumida y entendible.

## Valor del proyecto

TranxMex transforma datos financieros en decisiones prácticas. Además de mostrar el estado actual de la empresa, ayuda a detectar riesgos de caja, morosidad y pérdida de eficiencia, con una narrativa pensada para dirección, acreedores o accionistas.
